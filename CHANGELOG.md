# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-XX

### Added

- Initial release of Polybridge Python client
- `PolybridgeClient` class with full API support
- `fetch_market_catalog()` method for browsing markets
- `fetch_timeseries()` method for fetching historical data
- `fetch_nested_timeseries()` method for hierarchical data
- `fetch_up_or_down_options_timeseries()` method for options metrics
- `fetch_above_options_timeseries()` method for above options data
- `TimeseriesResult` dataclass for structured responses
- Automatic pandas DataFrame conversion
- Full type hints and documentation
- Example scripts and unit tests
